# YouTube Downloader

A sleek web application to download YouTube videos in your preferred quality.

## Features

- Paste any YouTube URL to analyze available formats
- View video thumbnail, title, channel, and duration
- Select from multiple quality options (1080p, 720p, 480p, etc.)
- Download progress tracking
- Clean, modern UI

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python app.py
```

3. Open http://localhost:5001 in your browser

## Usage

1. Paste a YouTube video URL
2. Click "Analyze" to fetch video information
3. Select your preferred quality
4. Click "Download"
5. Once complete, click the link to save the file

## Note

Downloaded videos are saved to the `downloads/` folder in the project directory.
